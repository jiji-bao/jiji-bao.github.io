addslashes()和stripslashes()：字符串转义

PHP 中使用 addslashes() 函数转义字符串。所谓字符串转义，就是在某些特殊字符前面加上转义符号\，这些特殊字符包括单引号’、双引号”、反斜线\与空字符NUL。
addslashes() 函数返回转义后的字符串。
stripslashes() 函数可以还原经 addslashes()转义的字符串。

count()：获取数组长度

PHP count() 函数用来获取数组长度，也即计算数组元素的个数。
另外，count() 函数还可以统计对象中的属性个数。

参数说明：
arr 表示要被统计的数组。
mode 表示统计模式，是一个可选参数，默认为 0，不会对多维数组中的所有元素进行统计。如果将 mode 设置为 COUNT_RECURSIVE 或者 1，count() 将会递归地对多维数组的所有元素进行统计。
返回值：返回数组元素的个数，或者对象属性的个数。如果 arr 是其它类型的值（不是数组或者对象），比如是整数、字符串等，count() 函数将返回 1。如果 arr 是一个没有定义的变量，则返回 0。

数组排序

与数组排序相关的函数
函数 说明
sort() 对数组元素进行升序排序（从小到大）。
rsort() 对数组元素进行降序排序（从大到小）。
asort() 对数组元素进行升序排序（从小到大），并保持索引关系。
arsort() 对数组元素进行降序排序（从大到小），并保持索引关系。
shuffle() 对数组元素进行随机排序，也就是随机打乱数组。

each() 和 list() 函数
